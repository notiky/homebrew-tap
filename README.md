# Notiky Homebrew Tap

Official [Homebrew](https://brew.sh) tap for installing the [Notiky CLI](https://github.com/notiky/notiky-cli).

[Notiky](https://notiky.com) is the product engineering brain — the AI workspace where products are built, from first idea to scale.

## Install

```bash
brew install notiky/tap/notiky
```

Then run:

```bash
notiky setup
```
